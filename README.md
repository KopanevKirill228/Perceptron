```python
from sklearn.datasets import make_classification
```

```python
X, y = make_classification(
```

3. Среднее и стандартное отклонение считались только по обучающей выборке.

Формула стандартизации:

```text
z = (x - mean) / std
```

Такой подход предотвращает утечку данных из тестовой выборки в обучение.

## 2. Реализация однослойного перцептрона

Был реализован класс `SingleLayerPerceptron`.

Класс включает:

- инициализацию весов;
- sigmoid-функцию;
- прямой проход `forward`;
- вычисление binary cross-entropy loss;
- mini-batch обучение;
- предсказание класса по порогу `0.5`;
- сохранение train и validation loss по эпохам.

Основная модель имеет вид:

```text
z = wᵀx + b
ŷ = sigmoid(z)
```

Функция активации:

```text
sigmoid(z) = 1 / (1 + exp(-z))
```

Функция потерь:

```text
BCE = -mean(y * log(ŷ) + (1 - y) * log(1 - ŷ))
```

## 3. Обучение и визуализация

Модель была обучена с гиперпараметрами:

| Параметр | Значение |
|---|---:|
| Learning rate `η` | 0.1 |
| Количество эпох | 100 |
| Batch size | 32 |

После обучения были построены:

- график изменения train loss;
- график изменения test/validation loss;
- разделяющая граница `wᵀx + b = 0`;
- визуализация данных на плоскости.

Для построения разделяющей прямой использовалась формула:

```text
w1 * x1 + w2 * x2 + b = 0
```

Отсюда:

```text
x2 = -(w1 * x1 + b) / w2
```

При `η = 1.0` точность немного снизилась, так как слишком большой шаг может приводить к менее стабильным обновлениям.

#### Вывод

Наиболее удачным вариантом среди рассмотренных значений можно считать `η = 0.5`, так как он дал хорошую точность на train и test.

## Дополнительное задание 2. Hinge loss и L2-регуляризация

### 2.1. Сравнение BCE и Hinge loss

| Loss function | Train accuracy | Test accuracy | \|\|w\|\| | Final loss |
|---|---:|---:|---:|---:|
| BCE | 0.8718 | 0.8792 | 3.3653 | 0.3335 |
| Hinge | 0.8889 | 0.9262 | 2.6172 | 0.3045 |

### Анализ

Binary cross-entropy использует sigmoid-выход и метки `{0, 1}`.  
Hinge loss использует линейный выход `wᵀx + b` и метки `{-1, +1}`.

В данном эксперименте Hinge loss показала более высокую точность на тестовой выборке:

```text
BCE test accuracy = 0.8792
Hinge test accuracy = 0.9262
```

Также Hinge loss дала меньшую норму весов.

### Вывод

Для данного набора данных Hinge loss оказалась эффективнее BCE. Это может быть связано с тем, что Hinge loss ориентирована на увеличение зазора между классами.

---

### 2.2. L2-регуляризация

| λ | Train accuracy | Test accuracy | \|\|w\|\| | Final loss |
|---:|---:|---:|---:|---:|
| 0.0 | 0.8718 | 0.8792 | 3.3653 | 0.3335 |
| 0.001 | 0.8718 | 0.8792 | 3.2560 | 0.3381 |
| 0.01 | 0.8718 | 0.8792 | 2.5976 | 0.3689 |
| 0.1 | 0.8689 | 0.8725 | 1.2296 | 0.4854 |
| 1.0 | 0.8632 | 0.8792 | 0.2852 | 0.6378 |

### Анализ

При увеличении `λ` норма весов уменьшается:

```text
λ = 0.0   -> ||w|| = 3.3653
λ = 0.001 -> ||w|| = 3.2560
λ = 0.01  -> ||w|| = 2.5976
λ = 0.1   -> ||w|| = 1.2296
λ = 1.0   -> ||w|| = 0.2852
```

Это подтверждает, что L2-регуляризация ограничивает рост весов.

Малые значения `λ = 0.001` и `λ = 0.01` уменьшили норму весов без потери accuracy.  
При `λ = 0.1` точность немного снизилась.  
При `λ = 1.0` loss сильно вырос, что указывает на слишком сильную регуляризацию и признаки недообучения.

### Вывод

Наиболее сбалансированным вариантом можно считать `λ = 0.01`, так как оно уменьшило норму весов без потери accuracy.

## Дополнительное задание 4. Momentum

Был реализован градиентный спуск с моментом и проведено сравнение с обычным mini-batch SGD.

### Результаты

| Optimizer | Train accuracy | Test accuracy | Final train loss | Final test loss |
|---|---:|---:|---:|---:|
| SGD | 0.8718 | 0.8792 | 0.3164 | 0.3335 |

### Анализ

Обычный SGD показал test accuracy `0.8792`.

Momentum с `β = 0.99` показал лучшую test accuracy:

```text
Test accuracy = 0.8926
Final test loss = 0.3285
```

При этом train accuracy стала немного ниже, что может говорить о лучшем обобщении.

### Вывод

В данном эксперименте лучшим оказался momentum с `β = 0.99`. Он дал более низкий test loss и более высокую точность на тестовой выборке. Однако слишком большое значение `β` в общем случае может приводить к колебаниям, поэтому momentum нужно подбирать экспериментально.
